Swing and Survive

Game Description

Swing and Survive is a 3D, third person, action combat game where you try to defeat the boss monster that is spawned. CAUTION: There are animation locks so be careful on what skills you want to use!

Controls

The controls are as follows.

To move: Use A, S, D, W

To attack: Use hotkeys 1, 2, and 3

To dodge: Use spacebar
